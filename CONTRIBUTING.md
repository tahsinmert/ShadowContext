# Contributing to ShadowContext

Thank you for your interest in contributing to ShadowContext! We welcome contributions from the community to help make this the best AI context gatherer for macOS.

## How to Contribute

### Reporting Bugs
If you find a bug, please open an issue using the [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md). Include as much detail as possible, such as your macOS version, Xcode version, and steps to reproduce the issue.

### Suggesting Features
Have an idea for a new feature? We'd love to hear it! Open an issue using the [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.md).

### Submitting Pull Requests
1.  **Fork the repository** and create your branch from `main`.
2.  **Environment Setup**: Ensure you have Xcode 15.0+ installed. Follow the [Xcode Setup Guide](ShadowContext/setup_guide.md) to configure the project.
3.  **Code Style**: Follow standard Swift API Design Guidelines.
4.  **Commit Messages**: Use descriptive commit messages (e.g., `feat: add token counting logic`, `fix: resolve memory leak in FolderWatcher`).
5.  **Tests**: If applicable, add unit tests for your changes.
6.  **Submit the PR**: Link it to any related issues and provide a clear description of your changes.

## Development Setup

1.  Clone your fork: `git clone https://github.com/your-username/ShadowContext.git`
2.  Open `ShadowContext.xcodeproj` in Xcode.
3.  Run the app (`Cmd + R`) to ensure everything is working correctly.

## Code of Conduct
By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).
